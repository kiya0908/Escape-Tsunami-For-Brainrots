# Escape Tsunami For Brainrots | Unofficial Wiki & Tools

The ultimate resource for survival. Master the waves, optimize your base income, find every radioactive coin, and become the top brainrot collector.

**网站地址：https://escapetsunamiforbrainrots.space**

## 项目概述

这是一个为 **Escape Tsunami For Brainrots（ETFB）** - Roblox游戏开发的非官方维基和工具网站。ETFB是一款生存类游戏，玩家需要在不断上涨的海啸浪潮中建造基地、收集资源、寻找放射性硬币并成为顶级收集者。

## 功能特性

- **生存指南** - 掌握如何在海啸浪潮中生存
- **基地优化** - 最大化你的基地收益
- **资源收集** - 每一个放射性硬币的位置
- **脑干收集** - 成为顶级收集者
- **游戏代码** - 最新的兑换代码
- **多语言支持** - 支持英语、菲律宾语、越南语、印尼语、德语

## 技术栈

- **Next.js** - React 框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式框架
- **shadcn/ui** - UI 组件库

## 本地开发

### 安装依赖

```bash
npm install
```

### 运行开发服务器

```bash
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000) 查看结果。

### 构建生产版本

```bash
npm run build
npm start
```

## 项目结构

```
escapeysunami/
```
